🚀 Java Array Practice (DSA)

This repository contains simple Java solutions to common array-based problems.
It is created for practice and basic understanding of array concepts.

* Missing Number
* Rotate Array
* Move Zeroes
* Find Duplicates
* Majority Element(n/3)
* Maximum Subarray Sum
* Two Sum
* Count Occurrences
* Triplets with Zero Sum

* Basic DSA concepts (Arrays, Loops)
